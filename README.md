# IMS website

Corporate website for IMS Consultancy, deployed via Vercel.

**Live:** https://ims.surfpunkian.com
**Production (future):** https://ims.ai

## Editing content
1. Edit the relevant `.md` file in `/content`.
2. Open a pull request.
3. Vercel will post a preview URL as a comment on the PR within ~30 seconds.
4. Once reviewed, merge to `main` — production updates automatically.

## Editing design / styling
- Design tokens: `/styles/tokens.css`
- Component styles: `/styles/components.css`
- Same PR-preview flow as content.
